# Radon Guide v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the single-page, credibility-first Radon Guide educational site for South Lake Tahoe, content-driven and test-first.

**Architecture:** Next.js 16 (App Router) statically renders one long page assembled from presentational components fed by a typed content layer (`content/` → `lib/content.ts`, the only seam). Two forms use React 19 Server Actions (Resend). Every on-page statistic links to its primary source, enforced by a data-driven Playwright test. Built test-first (Playwright red→green→refactor).

**Tech Stack:** Next.js 16, React 19.2, TypeScript, Tailwind v4, shadcn/ui (Radix), Zod, Resend, Vercel Web Analytics, `@playwright/test`, `next/font` (Newsreader + Public Sans). Turbopack is the default dev/build engine.

**Spec:** `docs/superpowers/specs/2026-06-13-radon-guide-v1-website-design.md` — read it before starting; section references below (e.g. "spec §7.1") point to it.

### Task 2.2: Content schema (Zod) with the primary-tier invariant

**Files:** Create `content/schema.ts`.

- [ ] **Step 1: Write the schema**

Create `content/schema.ts`:
```typescript
import { z } from 'zod';
import { SOURCES, type SourceId } from './sources';

const sourceId = z.custom<SourceId>((v) => typeof v === 'string' && v in SOURCES, 'unknown sourceId');

export const StatSchema = z.object({
  id: z.string(),
  value: z.string(),       // display string, e.g. "≈50%"
  label: z.string(),
  sourceId,
});

export const SectionSchema = z.object({
  id: z.string(),
  title: z.string(),
  order: z.number().int(),
  body: z.string(),        // Markdown
});

export const TestingRouteSchema = z.object({
  status: z.string(),
  action: z.string(),
  primary: z.boolean().optional(),
});

export const MitigationRowSchema = z.object({
  system: z.string(),
  foundation: z.string(),
  cost: z.string(),
  reduction: z.string(),
  sourceId,
});

export const PageContentSchema = z.object({
  hero: z.object({
    eyebrow: z.string(),
    headline: z.string(),
    body: z.string(),
    caveat: z.string(),
    sourceIds: z.array(sourceId).min(1),
  }),
  stats: z.array(StatSchema),
  sections: z.array(SectionSchema),
  testingRoutes: z.array(TestingRouteSchema),
  mitigationRows: z.array(MitigationRowSchema),
});

export type PageContent = z.infer<typeof PageContentSchema>;

// Invariant: every stat (and mitigation row) cited on-page must be primary-tier.
export function assertPrimaryTier(content: PageContent): void {
  const offenders: string[] = [];
  for (const s of content.stats) if (SOURCES[s.sourceId].tier !== 'primary') offenders.push(`stat:${s.id}`);
  for (const m of content.mitigationRows) if (SOURCES[m.sourceId].tier !== 'primary') offenders.push(`mitigation:${m.system}`);
  if (offenders.length) throw new Error(`Non-primary sources on-page: ${offenders.join(', ')}`);
}
```

Note: `angi_2026` is `secondary`, so any stat citing it will (correctly) fail the invariant — mitigation *costs* cite EPA; Angi is referenced only in prose, never as a stat's `sourceId`.

'use client';
import { useEffect, useRef } from 'react';
import { track } from '@/lib/analytics';

export function FooterBeacon() {
  const ref = useRef<HTMLDivElement>(null);
  useEffect(() => {
    const el = ref.current; if (!el) return;
    const obs = new IntersectionObserver((es) => { if (es[0].isIntersecting) { track({ name: 'reached_footer' }); obs.disconnect(); } });
    obs.observe(el); return () => obs.disconnect();
  }, []);
  return <div ref={ref} aria-hidden />;
}
